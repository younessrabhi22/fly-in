## About Me

I'm **Youness**, a software developer and student at **1337 Coding School** (42 Network) in Rabat, Morocco — a peer-to-peer engineering program with no instructors, no lectures, and no traditional courses. Everything is built from scratch, evaluated by peers.

My work spans low-level systems programming in C, application development in C++ and C#, and Python projects built during the 1337 curriculum. I'm comfortable working across paradigms — functional, object-oriented, and procedural — and I focus on understanding how things work under the hood rather than relying on abstractions.

- Working in **C**, **Python**, **C++**, and **C#**
- Interested in systems programming, algorithms, and clean software architecture
- Open to **internship opportunities** in software engineering
